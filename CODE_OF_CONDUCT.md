# Code of Conduct

This project adopts the
**[Contributor Covenant Code of Conduct, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)**.

The full canonical text is published at the link above. By participating in
this project (filing issues, opening PRs, commenting), you agree to abide
by it.

## In short

- Be respectful. Disagree about ideas, not people.
- No harassment, discriminatory language, or personal attacks.
- Assume good faith; ask before assuming malice.
- If a contributor's behavior makes the project worse for others,
  maintainers may remove their contributions or block them.

## Reporting

Email **conduct@haruspex.guru** to report a problem. Reports are handled
confidentially.

## Scope

This Code of Conduct applies within all project spaces — issues, pull
requests, discussions, and any official communication channel — and also
applies when an individual is officially representing the project in public
spaces.

> **Maintainer note:** if a future content filter blocks the verbatim text
> of the Contributor Covenant, this short summary suffices and the canonical
> text is reachable via the link above.
